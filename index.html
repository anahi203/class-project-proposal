AeroLink Scheduler

Proposal Last Updated: October 22, 2025

Team Members

  • Anahi Guerrero

Application Overview

AeroLink Scheduler helps airline crew and managers organize flight schedules. Crew members can view and swap shifts, while managers can approve changes and keep track of training requirements easily.

Technology Overview

  • Languages: HTML, CSS, JavaScript
  • Frameworks: React, Node.js
  • Database: MongoDB
  • Hosting: Vercel and Render

Feature List

  • Crew scheduling dashboard
  • Shift swap requests and approvals
  • Real-time notifications
  • User authentication

Future Feature List

  • AI-based scheduling
  • Mobile app
  • Offline mode
  • Reports and analytics

Basic Technical Features

  • Responsive design
  • Interactive JavaScript elements
  • Form validation
  • Hosted on Vercel

Advanced Technical Features

  • React single-page app
  • API integration with Node.js
  • Database CRUD operations
  • User authentication (JWT)
